# DJ Szambiarz — strona WWW

Oficjalna strona internetowa **DJ Szambiarz**.

Lekka, statyczna strona HTML — bez kroku budowania (*no build step*). Hostowana na GitHub Pages.

## 📁 Struktura projektu

```text
.
├── index.html        # strona główna
├── assets/
│   ├── img/          # grafiki
│   ├── css/          # style
│   └── js/           # skrypty
└── README.md
```

> ℹ️ Kod strony zostanie dodany w kolejnym kroku.

## 🚀 Uruchomienie lokalne

Otwórz `index.html` w przeglądarce lub uruchom prosty serwer HTTP:

```bash
python3 -m http.server 8000
# następnie otwórz: http://localhost:8000
```

## 🌐 Wdrożenie (GitHub Pages)

Po dodaniu `index.html` opublikuj stronę z gałęzi `main`:
**Settings → Pages → Source: *Deploy from a branch* → `main` / `(root)`**

Adres docelowy:
`https://m4tr1x-dev.github.io/dj-szambiarz-website/`
